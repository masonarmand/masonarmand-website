Cat's Cradle

Cat's Cradle is a game I made in about one day for my Experimental Game Jams website. Cat's Cralde is written in C and uses the Raylib game library. The theme for the jam was loop, so I made a game where you have to 'loop' around mice with your tail to capture them.
